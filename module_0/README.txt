Проявил инициативу и слегка модифицировал программу. Вначале необходимо
ввести число, до какого компьютер может загадывать (для задания необходимо 
ввести 100). 
С математической точки зрения всё работает корректно. Поясняю. 
Максимальное количество попыток будет зависить от того, два в какой степени
является больше загаданного числа. В случае числа 100 больше является 2**7,
т.е. максимум попыток может быть 7.  Для числа 1000 будет 2**10 (1024), 
т.е. максимум может быть 10 попыток. И чем больше число, тем ближе среднее 
количество попыток будет к максимальному количеству попыток.
